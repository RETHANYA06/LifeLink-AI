# 📖 Overview

LifeLink AI is a next-generation healthcare and emergency coordination platform designed to reduce emergency response time using Artificial Intelligence, real-time communication, and smart healthcare resource management.

The platform integrates patients, doctors, hospitals, ambulances, blood donors, and emergency administrators into a unified ecosystem where AI assists in emergency decision-making, hospital selection, doctor recommendations, and healthcare coordination.

---

# 🎯 Problem Statement

Medical emergencies often suffer from:

- 🚑 Ambulance delays
- 🏥 Unknown hospital bed availability
- 👨‍⚕️ Difficulty finding the right doctor
- 🩸 Blood shortages
- 📞 Manual emergency coordination
- ⏱ Critical time loss during the "Golden Hour"

LifeLink AI addresses these challenges by creating a centralized intelligent healthcare network capable of making real-time decisions during emergencies.

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/LifeLink-AI.git
```

```bash
cd LifeLink-AI
```

---

## Frontend

```bash
cd client
npm install
npm run dev
```

---

## Backend

```bash
cd server
npm install
npm run dev
```

---

## AI Service

```bash
cd ai-service
pip install -r requirements.txt
uvicorn main:app --reload
```
